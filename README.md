# OpenIpico

**Open source timing software for IPICO Sports RFID timing readers.**

> This project aims to build an open, community-driven alternative to the proprietary iPicoWiz software for managing IPICO Sports timing hardware at triathlons and running events.

## Supported Hardware

| Reader | Ports | Status |
|--------|-------|--------|
| IPICO Super Elite Reader | 10000, 10200, 10201 | ✅ Tested |
| IPICO Elite Reader | 10000, 10200, 10201 | ✅ Expected |
| IPICO Lite Reader | 10000, 10200, 10201 | ✅ Expected |
| IPICO Lite+ Reader | 10000, 10200, 10201 | ✅ Expected |

## Features

- [x] Connect to IPICO readers via TCP/IP (Ethernet/UTP)
- [x] Receive raw tag stream (port 10000)
- [x] Receive First-Seen / Last-Seen data (port 10200)
- [x] Parse IP-X protocol tag records
- [x] Generate HTML results page from QuickExport.csv
- [ ] Split/lap time calculation
- [ ] Live timing dashboard
- [ ] Tag pass adjustment (shift a reading to the correct timing point)
- [ ] Race registration integration
- [ ] Results export (CSV, JSON, PDF)
- [ ] Docker deployment

## Quick Start

### 1. Connect to a reader and stream tag data

```bash
python3 reader_client.py --host <reader-ip> --port 10200
```

### 2. Generate HTML results from QuickExport.csv

```bash
python3 results_page.py QuickExport.csv --output uitslag.html
```

Open `uitslag.html` in any browser — includes overall rankings, per-category results, and search.

## How It Works

```
IPICO Reader (hardware)
    │
    │ TCP/IP over Ethernet/UTP
    │
Tag passes → IP-X stream (aa0580a01f5271...)
    │
    ├─ TagMap (chip → bib number)
    │
    └─ Startlijst (bib → name, club, category)
            │
            ▼
    Race Results (bib + name + club + times)
            │
            ▼
    HTML page (uitslag.html) ← results_page.py
```

## The Data Chain

| File | Purpose |
|------|---------|
| **TagMapYYYY_vN.csv** | Maps RFID chip ID → bib number (e.g. `0580a01f5271` → `1`) |
| **Startlijst.xlsx** | Maps bib number → athlete details (name, club, category) |
| **QuickExport.csv** | Race results with all timing data (from iPicoWiz or OpenIpico) |
| **uitslag.html** | Final results page (generated by `results_page.py`) |

## Roadmap

1. **Tag pass adjustment** — Manually shift a misread tag to the correct timing point (key missing feature from iPicoWiz)
2. **Live timing page** — Real-time results during the race
3. **Registration import** — Load startlijsten directly
4. **Docker deployment** — Run on a Raspberry Pi at the race location
5. **Multiple reader support** — Combine Super Elite + Lite readers for longer timing lines
